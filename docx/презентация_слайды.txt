Тема диплома: «Модуль учёта волонтёрских часов».
Условное обозначение: «VolonterHoursAPP 1.0».
Автор: Макаров Вячеслав Васильевич, группа 225, Зеленодольский механический колледж.
Специальность: 09.02.07 «Информационные системы и программирование».

Что это: настольное приложение для учёта волонтёрских часов.
Технологии: Java 22, JavaFX (десктоп, НЕ веб), PostgreSQL, IntelliJ IDEA.
Пароли: SHA-256 (не открытым текстом). Экспорт отчётов: CSV (совместим с Excel). PDF нет.
Роли: Администратор (всё) / Куратор (рабочие разделы, без «Пользователи») /
      Пользователь (только просмотр: дашборд, мероприятия, инфо о волонтёрах, рейтинг, настройки).

База данных — 6 таблиц:
  organizers (id, name) — справочник организаторов
  volunteers (id, fio, birth_date, phone, email, registered_at)
  events (id, title, datetime, location, type, organizer_id → organizers)
  users (id, username, password_hash, full_name, role, active, created_at)
  participations (id, volunteer_id → volunteers, event_id → events, hours_worked,
                  confirmed, confirmed_by_user_id → users, confirmed_at)
  audit_log (id, occurred_at, actor_user_id → users, action, details)
Связи: volunteers 1—∞ participations (CASCADE), events 1—∞ participations (CASCADE),
       organizers 1—∞ events (RESTRICT), users 1—∞ participations/audit_log (SET NULL).

Разделы приложения: Дашборд, Волонтёры, Мероприятия, Участия, Инфо о волонтёрах,
  Аналитика, Рейтинг, План мероприятий, Журнал действий, Пользователи, Настройки, Справка.
Ключевая логика: только ПОДТВЕРЖДЁННЫЕ часы идут в статистику и рейтинг.
Тестирование: 11 тест-кейсов пройдено.
